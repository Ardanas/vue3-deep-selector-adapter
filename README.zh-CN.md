# Vue 3 深度选择器适配器

[English](./README.md)

Vue 3 深度选择器适配器是一个工具,旨在帮助将 Vue 2 的深度选择器迁移到与 Vue 3 兼容的语法.

## 特性

- 将 Vue 2 的深度选择器 (>>>, /deep/, ::v-deep) 转换为 Vue 3 的 :deep() 语法
- 支持 .vue, .css, .scss, .sass, .less 和 .styl 文件
